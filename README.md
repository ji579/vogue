# vogue

프로젝트 개요

VOGUE 웹사이트를 참고해

메인 페이지를 포함한 다중 페이지 구조로 화면을 구현한

웹 퍼블리싱 프로젝트입니다.


콘텐츠 성격에 따라 페이지별 레이아웃과 마크업 구조를 분리해 설계했으며,

공통 요소를 일관되게 적용해 사이트 전체의 흐름을 유지했습니다.


HTML 시맨틱 구조와 CSS 레이아웃을 기반으로,

페이지 구성과 퍼블리싱 완성도에 중점을 두어 제작했습니다.



실현 기능

	•	메인 페이지를 포함한 다중 페이지 웹사이트 구조 구현

	•	페이지별 콘텐츠 성격에 맞춘 레이아웃 및 마크업 구성

	•	공통 헤더·푸터 구조 적용을 통한 사이트 통일성 유지

	•	페이지 이동 시 구조가 자연스럽게 이어지도록 네비게이션 구성

	•	각 페이지에 맞는 인터랙션 및 시각 효과 적용





사용 기술

	•	HTML5 : 시맨틱 태그를 활용한 구조 설계

	•	CSS3 : Flexbox를 활용한 레이아웃 구성 및 스타일링

	•	JavaScript : 기본적인 인터랙션 구현

	•	AI Coding 활용 : 마크업 구조 및 코드 개선 보조
